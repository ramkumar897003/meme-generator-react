# Meme Generator in React

This app will generate meme by selecting available image and by adding text in image.

### Installing

A step by step series of examples that tell you how to get app running

### Using npm

```
npm install
```

```
npm start
```


### Using yarn

```
yarn
```

```
yarn start
```
